## 开始
```
yarn 
yarn start // 起项目
yarn build // 打包
yarn analyzer //查看分析文件
```


## 目录结构

```
project
│   
└─── config 配置文件 
│ 
└─── build 打包文件
│ 
└─── src   业务文件
      │ 
      └─── assets 图片等静态资源
      │   
      └─── components 公共组件 
      │ 
      └─── pages  页面
      │ 
      └─── styles 样式文件
      │ 
      └─── app.tsx 入口，这里为第一层路由页
```
